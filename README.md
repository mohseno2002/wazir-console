# لوحة قيادة الوزير

منظومة موحّدة لتطبيقات التحوّل الرقمي لقطاع الموارد المائية والري، تفتح كل منظومة داخلياً بملء الشاشة، وتُثبّت على الموبايل كتطبيق (PWA).

## المحتويات
- `index.html` — اللوحة
- `manifest.webmanifest` — بيانات التثبيت
- `sw.js` — التخزين المؤقت والعمل دون اتصال
- `icon-192.png` / `icon-512.png` / `icon-maskable-512.png` — الأيقونات

## الرفع على GitHub Pages
1. أنشئ مستودعاً جديداً باسم `wazir-console`.
2. ارفع كل الملفات في الجذر (مش داخل مجلد فرعي).
3. من Settings ← Pages اختر الفرع `main` والمجلد `/ (root)`.
4. الرابط سيكون: `https://mohseno2002.github.io/wazir-console/`
5. افتحه من المتصفح ← اضغط "إضافة إلى الشاشة الرئيسية" لتثبيته كأيقونة.

## ملاحظات
- التطبيقات الستة تفتح داخل iframe. لو أي تطبيق رفض الفتح بالداخل، استخدم زر "نافذة" بالكارت.
- لتحديث الأيقونة بعد أي تعديل، ارفع `sw.js` بعد تغيير رقم النسخة `wazir-console-v1`.
