# لوحة قيادة الوزير | Hydro Command Center

مركز قيادة هيدروليكي رقمي موحّد لقطاع الموارد المائية والري. يضم خريطة حية للنيل والترع، مؤشرات لحظية، ولوحة تنبيهات، ويفتح كل منظومة داخلياً بملء الشاشة. يُثبّت على الموبايل كتطبيق (PWA).

## وضعان للعرض
- **ليلي** (افتراضي): مركز قيادة سيان نيون على خلفية كحلية.
- **نهاري**: أسلوب «المخطط الهندسي» (Blueprint) — ورق فاتح وحبر كحلي ولمسات ذهبية.
يُبدّل بالزر أعلى الترويسة، ويُحفظ الاختيار تلقائياً، ويتبع تفضيل النظام أول مرة.

## المحتويات
- `index.html` — اللوحة
- `manifest.webmanifest` — بيانات التثبيت
- `sw.js` — التخزين المؤقت والعمل دون اتصال
- `icon-192.png` / `icon-512.png` / `icon-maskable-512.png` — الأيقونات

## الرفع على GitHub Pages
1. أنشئ مستودعاً باسم `wazir-console`.
2. ارفع كل الملفات في الجذر (مش داخل مجلد فرعي).
3. Settings ← Pages ← الفرع `main` والمجلد `/ (root)`.
4. الرابط: `https://mohseno2002.github.io/wazir-console/`
5. افتحه ← «إضافة إلى الشاشة الرئيسية» لتثبيته كأيقونة.

## ملاحظات
- المنظومات الست تفتح داخل iframe؛ لو رفض أحدها الفتح بالداخل استخدم زر «نافذة».
- لتحديث الكاش بعد أي تعديل، غيّر رقم النسخة `wazir-console-v1` في `sw.js`.
