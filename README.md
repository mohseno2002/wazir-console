# لوحة قيادة الوزير | Hydro Command Center

مركز قيادة هيدروليكي رقمي موحّد. وضعان (ليلي/نهاري)، يفتح المنظومات داخلياً، ويُثبّت كتطبيق (PWA). يحتوي وكيل تقرير يومي يعمل **داخل المتصفح** (لا حاجة لخوادم أو GitHub Actions).

## الملفات
- index.html, manifest.webmanifest, sw.js, الأيقونات الثلاث

## تقرير اليوم
بطاقة "تقرير اليوم" تولّد ملخصاً تلقائياً من مصادر بياناتك عند فتح اللوحة، وزر "↻ تحديث" لإعادة التوليد. آخر تقرير يُحفظ محلياً ويظهر فوراً عند الفتح.

### ربط مصادرك
افتح index.html وابحث عن `REPORT_SOURCES` في الجزء البرمجي. لكل منظومة:
- **Firebase**: ضع `firebaseUrl` (مثل https://xxx-default-rtdb.firebaseio.com) ومسار كل مؤشر في `path`.
- **Google Sheets**: انشُر الشيت كـ CSV وضع الرابط في `sheetCsvUrl` واسم العمود في `column`.
- اضبط `thresholds` (warn/danger) لتلوين الحالة.
المؤشرات بدون مصدر تظهر "لا بيانات" دون أن تكسر التقرير.

## الرفع على GitHub Pages
ارفع كل الملفات في جذر مستودع `wazir-console`، ثم Settings ← Pages ← main / root.
الرابط: https://mohseno2002.github.io/wazir-console/

## ملاحظات
- التقرير يعمل عبر HTTP (الموقع المنشور)؛ قد لا يعمل عند فتح الملف محلياً بـ file://.
- لتثبيته كتطبيق: افتح الرابط ← "إضافة إلى الشاشة الرئيسية".
